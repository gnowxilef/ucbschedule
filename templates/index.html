UCB Schedule of Classes

API

We provide an API for data obtained from scraping the UC Berkeley Schedule of Classes. Currently, data from Spring 2014 (starting from 2013-10-21) is in the database. Examples:

  • /api/departments?page=1: returns first page of departments
  • /api/departments/35: returns department with id=35 (COMPSCI)
  • /api/departments/35/courses?page=1: returns first page of courses with department COMPSCI
  • /api/courses/615: returns course with id=615 (COMPSCI 189, Spring 2014)
  • /api/sections/1854: returns section with id=1854 (COMPSCI 189 LEC 001, Spring 2014)
  • /api/sections/1854/instances: returns instances (where an instance is a section's data for a particular day) of COMPSCI 189 LEC 001, Spring 2014
